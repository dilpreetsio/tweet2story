# Tweet2Story
Create instagram stories from tweets using cli.


## Requirements 
* [Pillow](https://pillow.readthedocs.io/en/stable/) 
* [Tweepy](https://www.tweepy.org/) 
* [Click](https://click.palletsprojects.com/en/7.x/)
